# Project Context: Onsus eCommerce Clone

## Overview
This project is a functional clone of the [Onsus Electronics WooCommerce Storefront](https://preview.themeforest.net/item/onsus-electronics-ecommerce-nextjs-react-template/full_screen_preview/57752071). The goal is to replicate the premium aesthetics and rich functionality of the original template using a modern headless stack.

## Core Objectives
1.  **Visual Fidelity**: Match the high-end, responsive design of the Onsus template.
2.  **Performance**: Leverage Next.js 13 for fast page loads and SEO.
3.  **Dynamic Catalog**: Use Supabase for real-time product management and multi-language support.
4.  **Localization**: Support English (EN), Azerbaijani (AZ), and Russian (RU) across all storefront elements.

## Key Features to Implement
- [ ] **Phase 1: Supabase Integration**: Robust data fetching, authentication, and real-time updates.
- [ ] **Phase 2: Dynamic Storefront**: /shop page with filters, /products detail pages.
- [ ] **Phase 3: User Interaction**: Cart/Wishlist persistence, Comparison tool, User accounts.
- [ ] **Phase 4: Checkout & Orders**: Stripe/Mock payment integration, order tracking.
